### Harbor 报错处理

~~~shell
docker login https://www.harbor.org
Username: admin
Password:
Error response from daemon: Get https://www.harbor.org/v2/: x509: certificate signed by unknown authority

#要认证
cat > /etc/docker/daemon.json <<EOF
{
  "insecure-registries": [
    "www.harbor.org"
  ]
}
EOF

~~~
